Incheon Style Guide

1.SCROLL JQUERY

1.scrollHeight = $("body").height() = 엘리먼트의 높이

2.scrollRealHeight = (scrollHeight - $(window).height()); = 스크롤할 실제 거리

3.winScrollTop = $(window).scrollTop() = 스크롤바의 상단 위치

4.scrollPercent (winScrollTop / scrollRealHeight) * 100 = 백분율 (비율 x 100(%))

5.textPercent = Math.floor(scrollPercent) = 소수점 버리기

6.onTop = $(document).height() - $(window).height() - $('.footer').height(); = 스크롤이 문서하단에 도착했는지 계산
